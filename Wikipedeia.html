Способы активного отображения информации

Представленная в сети информация может быть доступна:

  • только для чтения(пассивно)
  • для чтения и добавления/изменения("активно ")

К спрособам активного отображения информации во Всемирной паутине относятся:

  • гостевые книги(англ. questbook);
  • форумы(англ. forum);
  • чаты(англ. blog)
  • wiki-проекты ;
  • социалиные сети(англ. social networking service);
  • системы управления контентом(англ.content managing system, CMS).
